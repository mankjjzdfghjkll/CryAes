## Description
Complete implementation of the Advanced Encryption Standard (AES/Rijndael) algorithm in C# with Windows Forms GUI.

## Features
- ✅ AES-128, AES-192, AES-256
- ✅ ECB, CBC, CTR modes
- ✅ File encryption/decryption
- ✅ Parallel processing (async/await)
- ✅ Windows Forms GUI
- ✅ NIST test vectors validation
